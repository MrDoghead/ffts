# Intro

This is a demo toolkit for fft-like transformations.

## FFT and IFFT

An python implementation of FFT: 

```bash
python fft.py
```

A test code for FFTW, which is implemented in C++:

```bash
g++ test_fftw.cpp -lfftw3 -lm
```

## NTT and INTT

An python implementation of NTT: 

```bash
python ntt.py
```
